# Imágenes de Productos

Esta carpeta contiene las imágenes para los productos del catálogo.

## Estructura de carpetas

```
images/
├── iphones/          # Imágenes de iPhones
└── macbooks/         # Imágenes de MacBooks
```

## Imágenes requeridas

### iPhones (public/images/iphones/)
- `iphone-11.jpg`
- `iphone-12-pro.jpg`
- `iphone-13.jpg`
- `iphone-14-pro-max.jpg`
- `iphone-15-pro.jpg`
- `iphone-16-pro-max.jpg`

### MacBooks (public/images/macbooks/)
- `macbook-air-m1.jpg`
- `macbook-air-m2.jpg`
- `macbook-pro-14.jpg`
- `macbook-pro-16.jpg`

## Recomendaciones

- **Formato**: JPG o PNG
- **Tamaño recomendado**: 800x800px o 1000x1000px
- **Peso**: Máximo 200KB por imagen para mejor rendimiento
- **Orientación**: Cuadrada o vertical
- **Fondo**: Blanco o transparente para mejor presentación

## Cómo agregar imágenes

1. Guarda las imágenes de productos en la carpeta correspondiente
2. Usa los nombres de archivo exactos listados arriba
3. Si agregas nuevos productos, actualiza el archivo `src/data/products.ts` con la ruta correcta

## Imagen de placeholder

Si una imagen no está disponible, la aplicación mostrará un placeholder automático.
